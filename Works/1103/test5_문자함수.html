length : 객체(문자열)의 길이

slice(a,b) : 지정된 문자열(a에서 b앞까지)을 추출해준다.

substring(a,b) : 지정된 문자열(a에서 b앞까지)을 추출해준다.

substr() :

replace(a,b) : a를 b로 치환

replaceAll(a,b) : a를 b로 모두 치환

toUpperCase() : 대문자로 치환

toLowerCase() : 소문자로 치환

concat(a, b)~~ : a와 b를 결합한다.

trim() : 양쪽 공백을 없애준다.

trimStart() : 앞쪽 공백을 없애준다.

trimEnd() : 뒤쪽 공백을 없애준다.

charCodeAt() : 그 자리 문자의 아스키 코드 값을 가져온다

split() : 구분자를 기준으로 잘라준다. 잘라진 각각의 요소는 배열로 저장

indexOf() : 특정 문자의 위치값가져옴

lastindexOf() : 뒤에서 부터 특정 문자의 위치값가져옴 단, 위치값은 인덱스 0번부터 몇번째인지 알려줌

search() : 특정문자의 위치값을 찾아준다.(찾는값이 여러개일경우 맨 앞글자의 인덱스 번호를 가져온다)

match() : 찾는 문자열이 있으면 '문자열'을 반환, 없으면 '공백'을 반환

includes() : 찾는 문자열을 포함하고있으면 true 없으면 false 반환
